What is a learning log?

Learning Log is a personal journal for recording what you`re learning about.

Why keep a learning log?

Because it helps you to learn more effectively. When you write about a topic, you`re forced to think about it more deeply and clearly. You make connections between ideas, spot patterns, and recognize concepts that you might otherwise have missed.
